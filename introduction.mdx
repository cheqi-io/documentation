## How It Works

```mermaid
sequenceDiagram
    participant Merchant as Your POS
    participant Cheqi as Cheqi API
    participant Customer as Customer App
    
    Merchant->>Cheqi: 1. Match customer (card/IBAN)
    Cheqi->>Merchant: 2. Return device keys
    Merchant->>Merchant: 3. Encrypt receipt
    Merchant->>Cheqi: 4. Send encrypted receipt
    Cheqi->>Customer: 5. Push notification
    Customer->>Customer: 6. Decrypt & view
```

## Quick Start

Get started in minutes with our Java SDK:

```java
// 1. Initialize SDK
CheqiSDK sdk = CheqiSDK.builder()
    .apiEndpoint(Environment.PRODUCTION)
    .apiKey(System.getenv("CHEQI_API_KEY"))
    .build();

// 2. Send receipt
ReceiptResult result = sdk.getReceiptService()
    .processCompleteReceipt(customer, receipt);

if (result.isSuccess()) {
```
