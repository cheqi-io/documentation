## The Receipt Flow (Language-Agnostic)

Regardless of which SDK you use, sending a receipt follows these steps:

<Steps>
  <Step title="Initialize SDK">
    Configure the SDK with your API key and environment (sandbox/production).
  </Step>
  
  <Step title="Identify Customer">
    Provide payment details from your POS:
    - **Card Payment**: Payment Account Reference (PAR) + Card Provider
    - **Bank Transfer**: IBAN
    - **Fallback**: Email address (for non-Cheqi users)
  </Step>
  
  <Step title="Build Receipt Data">
    Structure your receipt with:
    - Document number (invoice/receipt ID)
    - Issue date
    - Currency
    - Line items (products/services)
    - Tax breakdown
    - Total amounts
  </Step>
  
  <Step title="Send Receipt">
    Call the SDK's receipt processing method. The SDK handles:
    - Customer matching
    - UBL template generation
    - End-to-end encryption
    - Multi-device delivery
    - Email fallback (if needed)
  </Step>
  
  <Step title="Handle Response">
    Check the result:
    - **Success**: Receipt delivered to customer's devices
    - **Customer Not Found**: Prompt for email or handle as needed
    - **Error**: Log and retry with exponential backoff
  </Step>
</Steps>

## Quick Example (Java)

Here's a complete example using the Java SDK. For other languages, see the SDK-specific guides above.

### 1. Install SDK

<Tabs>
  <Tab title="Maven">
    ```xml
    <dependency>
        <groupId>io.cheqi</groupId>
        <artifactId>cheqi-sdk</artifactId>
        <version>1.0.3</version>
    </dependency>
    ```
  </Tab>
  
  <Tab title="Gradle">
    ```gradle
    implementation 'io.cheqi:cheqi-sdk:1.0.3'
    ```
  </Tab>
</Tabs>

### 2. Initialize SDK

```java
import com.cheqi.sdk.CheqiSDK;
import com.cheqi.sdk.config.Environment;

CheqiSDK sdk = CheqiSDK.builder()
    .apiEndpoint(Environment.SANDBOX)  // Use SANDBOX for testing
    .apiKey(System.getenv("CHEQI_API_KEY"))
    .build();
```

<Warning>
  Store your API key in environment variables, never hardcode it!
</Warning>

### 3. Identify Customer

```java
import com.cheqi.sdk.models.*;
import com.cheqi.commons.enums.CardProvider;

IdentificationDetails customer = IdentificationDetails.builder()
    .paymentType(PaymentType.CARD_PAYMENT)
    .cardDetails(CardDetails.builder()
        .paymentAccountReference("PAR123456789")  // From payment terminal
        .cardProvider(CardProvider.VISA)
        .build())
    .customerEmail("customer@example.com")  // Optional fallback
    .build();
```

<Info>
  **Payment Account Reference (PAR)** is provided by modern payment terminals. It's a privacy-friendly way to identify customers without storing card numbers.
</Info>

### 4. Build Receipt

```java
import java.math.BigDecimal;
import java.time.Instant;

ReceiptTemplateRequest receipt = ReceiptTemplateRequest.builder()
    .documentNumber("INV-2024-001")
    .issueDate(Instant.now())
    .currency("EUR")
    .totalAmount(new BigDecimal("12.10"))
    .totalTaxAmount(new BigDecimal("2.10"))
    
    // Add products
    .addProduct(Product.builder()
        .name("Espresso")
        .quantity(2.0)
        .unitCode(UnitCode.ONE)
        .unitPrice("5.00")
        .subtotal("10.00")
        .total("12.10")
        .addTax(21.0, "VAT", "2.10")
        .build())
    
    // Add tax breakdown
    .addTax(Tax.builder()
        .rate(21.0)
        .type("VAT")
        .taxableAmount("10.00")
        .amount("2.10")
        .label("VAT 21%")
        .build())
    
    .build();
```

### 5. Send Receipt

```java
import com.cheqi.sdk.receipt.ReceiptResult;

ReceiptResult result = sdk.getReceiptService()
    .processCompleteReceipt(customer, receipt);

if (result.isSuccess()) {
    System.out.println("‚úÖ Receipt delivered to " + result.getReceiptCount() + " devices");
    System.out.println("Receipt ID: " + result.getCheqiReceiptId());
} else if (result.isCustomerNotFound()) {
    System.out.println("‚ö†Ô∏è Customer not found");
} else if (result.isDeliveredViaEmail()) {
    System.out.println("üìß Receipt sent to: " + result.getEmailAddress());
} else {
    System.out.println("‚ùå Failed: " + result.getMessage());
}
```

## Complete Example

Here's a complete working example:

```java
import com.cheqi.sdk.CheqiSDK;
import com.cheqi.sdk.config.Environment;
import com.cheqi.sdk.models.*;
import com.cheqi.sdk.receipt.ReceiptResult;
import com.cheqi.commons.enums.CardProvider;
import java.math.BigDecimal;
import java.time.Instant;

public class CheqiQuickstart {
    public static void main(String[] args) {
        // 1. Initialize SDK
        CheqiSDK sdk = CheqiSDK.builder()
            .apiEndpoint(Environment.SANDBOX)
            .apiKey(System.getenv("CHEQI_API_KEY"))
            .build();
        
        // 2. Identify customer
        IdentificationDetails customer = IdentificationDetails.builder()
            .paymentType(PaymentType.CARD_PAYMENT)
            .cardDetails(CardDetails.builder()
                .paymentAccountReference("PAR123456789")
                .cardProvider(CardProvider.VISA)
                .build())
            .customerEmail("customer@example.com")
            .build();
        
        // 3. Build receipt
        ReceiptTemplateRequest receipt = ReceiptTemplateRequest.builder()
            .documentNumber("INV-001")
            .issueDate(Instant.now())
            .currency("EUR")
            .totalAmount(new BigDecimal("12.10"))
            .totalTaxAmount(new BigDecimal("2.10"))
            .addProduct(Product.builder()
                .name("Espresso")
                .quantity(2.0)
                .unitCode(UnitCode.ONE)
                .unitPrice("5.00")
                .subtotal("10.00")
                .total("12.10")
                .addTax(21.0, "VAT", "2.10")
                .build())
            .addTax(Tax.builder()
                .rate(21.0)
                .type("VAT")
                .taxableAmount("10.00")
                .amount("2.10")
                .build())
            .build();
        
        // 4. Send receipt
        try {
            ReceiptResult result = sdk.getReceiptService()
                .processCompleteReceipt(customer, receipt);
            
            if (result.isSuccess()) {
                System.out.println("‚úÖ Success! Receipt delivered.");
            }
        } catch (Exception e) {
            System.err.println("Error: " + e.getMessage());
        }
    }
}
```

## What Happens Next?

**Cheqi handles everything for you:**

<Steps>
  <Step title="Customer Matching">
    Cheqi matches the customer using the payment details (PAR, IBAN, etc.)
  </Step>
  
  <Step title="Receipt Generation">
    A UBL-compliant receipt template is generated from your data
  </Step>
  
  <Step title="Smart Delivery Decision">
    **If customer uses Cheqi:**
    - Receipt is encrypted for each device using hybrid RSA+AES encryption
    - Pushed to all customer devices with notification
    - Only customer can decrypt and view
    
    **If customer doesn't use Cheqi:**
    - We automatically generate a professional PDF
    - We send it via email (if you provided one)
    - Beautifully formatted, no work needed from you
  </Step>
  
  <Step title="Delivery Confirmation">
    You receive confirmation of successful delivery (app or email)
  </Step>
</Steps>

<Info>
  **Complete solution:** You send the receipt data once. We handle customer matching, encryption, PDF generation, email delivery, and multi-device sync. Focus on your business, we handle the receipts.
</Info>
